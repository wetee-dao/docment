# DAO 概述

WeTEE 正在通过其集成的 DAO（去中心化自治组织）向完全去中心化的治理模型演进。DAO 允许社区成员和利益相关者参与平台的决策过程。

## 治理结构

*   **理事会:** 由选出的成员组成，负责监督平台并提出重大变更建议。
*   **代币持有者:** WETEE 原生代币持有者可以对提案进行投票并参与治理。
*   **民主机制:** 用于关键平台更新和参数更改的直接投票机制。

## 如何参与

任何持有 WETEE 代币的人都可以通过以下方式参与 DAO：
1.  **提交提案:** 提议新功能、预算分配或政策更改。
2.  **投票:** 对活跃提案进行投票，共同塑造平台的未来。
3.  **质押:** 质押代币以获得投票权并赚取奖励。
