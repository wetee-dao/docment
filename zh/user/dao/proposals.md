# 提案与投票

提案是 WeTEE DAO 内发起变更的主要方式。

## 创建提案

1.  **起草:** 定义提案的范围、预算和实施计划。
2.  **提交:** 使用 WeTEE dApp 提交您的提案。这通常需要支付少量的 WETEE 代币作为押金。
3.  **讨论:** 在论坛或社交频道与社区互动，解释您提案的优势。

## 投票流程

*   **投票期:** 每个提案都有预定义的投票窗口（例如 7 天）。
*   **信念投票:** 您锁定代币的时间越长，您的投票权就越大。
*   **结果:** 如果提案达到必要的法定人数和批准阈值，它将自动生效，或移交给技术委员会实施。
