# 设置钱包

PulsWeb2 运行在 Polkadot (波卡) 生态架构之上。使用 dApp 时，你可能需要一个兼容的钱包来管理账户并进行签名确认。

### 第一步：选择钱包
我们推荐使用以下波卡生态的主流钱包：
1.  **Polkadot{.js}** (浏览器插件 - 推荐)：功能最全。
2.  **Talisman** (浏览器插件)：界面美观，操作简单。
3.  **SubWallet** (浏览器插件/移动端)：综合体验极佳。

### 第二步：安装与创建
以 **Polkadot{.js}** 为例：
1.  前往 [Chrome 商店](https://chrome.google.com/webstore) 搜索并安装插件。
2.  点击“Create New Account” (创建新账户)。
3.  **极其重要**：抄写并安全保存你的 **12个助记词**。不要截图，不要存入云盘，这是找回你资产的唯一凭证。
4.  设置一个复杂的钱包密码。

### 第三步：连接到 PulsWeb2
1.  打开 PulsWeb2 dApp。
2.  点击右上角的 **“Connect Wallet”** (连接钱包)。
3.  在弹出的钱包窗口中允许 dApp 访问你的账户。
4.  连接成功后，你会看到你的地址与账户信息。

---

**小贴士**：PulsWeb2 使用的是波卡格式的地址。如果你在切换网络时发现地址变了，请不要惊慌，它们在底层是同一个账户。
