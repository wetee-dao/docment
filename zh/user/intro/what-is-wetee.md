# 什么是 PulsWeb2？

如果你是第一次接触 **PulsWeb2**，你可以把它想象成一个 **“自带保险箱的云服务器”**。

在传统的云服务（如阿里云、AWS）中，虽然你的数据是加密存储的，但在服务器运行（计算）你的程序时，内存中的数据对云服务商的管理员来说其实是“透明”的。

**PulsWeb2 改变了这一点。**

### 核心定义
PulsWeb2 是一个 **机密计算平台**。它利用区块链集成与硬件级别的“安全屋”（TEE，受信任执行环境），确保你的代码和数据在计算过程中也是加密的，连提供服务器的节点提供者也无法查看或篡改。

### 简单类比
*   **传统云服务**：你租了一个写字楼办公室，虽然门上有锁，但大楼管理员（云厂商）手里有万能钥匙，能随时进去翻看你的文件。
*   **PulsWeb2**：你在写字楼里放了一个**特制的保险柜**。你把代码和数据放进保险柜，保险柜自己在里面运行。即使管理员站在旁边，他也看不到保险柜里在干什么，更打不开它。

### 谁应该使用 PulsWeb2？
1.  **AI 开发者**：想保护辛苦训练的 AI 模型不被窃取。
2.  **Web3 创业者**：需要安全地生成和管理密钥（DKG），或者运行私密的智能合约。
3.  **金融/医疗机构**：处理高度敏感的隐私数据（如病例、交易记录），必须满足法律法规的隐私要求。
4.  **普通用户**：想运行一些不想被任何第三方监控的应用程序。

---

通过 PulsWeb2，你可以像使用普通云服务一样简单地部署应用，但却能获得顶级的隐私安全保障。
