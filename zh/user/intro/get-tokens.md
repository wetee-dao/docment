# 准备工作

在开始使用 PulsWeb2 之前，你需要完成简单的账户准备。

### 1. 设置钱包
PulsWeb2 目前基于波卡 (Polkadot) 生态技术构建。虽然处于测试阶段，但你仍然需要一个账户地址来标识你的身份。

我们推荐使用：
*   **Polkadot{.js} Plus** (浏览器插件)
*   **Talisman** 或 **SubWallet**

### 2. 获取使用权限 / 配额
PulsWeb2 **不发币**，使用过程中不会要求你购买或领取平台代币。

在当前阶段（内测/预览/灰度），运行资源通常通过 **权限** 或 **配额** 方式发放：

*   **管理员开通**：由官方或项目管理员为你的账号开通权限/配额。
*   **邀请或白名单**：加入测试计划后自动获得配额。

### 3. 账户用途（当前可确认的部分）
你的账户地址通常用于：
*   **身份标识/签名**：证明你是该工作负载（服务/任务）的所有者，并用于操作确认。
*   **权限校验**：系统根据你的权限决定是否允许创建/修改/停止服务。

---

**注意**：如处于测试环境，请勿存放不可恢复的生产数据。具体规则以当前环境公告为准。
