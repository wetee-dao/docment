# 机密存储 (Confidential Storage)

机密存储为机密服务和任务提供静态加密存储解决方案。它确保即使物理存储介质遭到破坏，存储在磁盘上的敏感数据也能得到保护。

## 特性

*   **静态加密:** 数据在写入磁盘前自动加密，在授权的 TEE 应用程序读取时自动解密。
*   **持久化卷 (PVC):** 与 Kubernetes 的持久化卷声明集成，允许机密服务在重启后保持状态。
*   **硬件绑定密钥:** 加密密钥在 TEE 内部管理，并可以绑定到硬件的信任根。

## 使用方法

在 WeTEE 中，您可以通过在应用部署配置中指定存储需求来请求机密存储。系统将自动配置一个加密卷并将其挂载到您的 TEE 容器中。
