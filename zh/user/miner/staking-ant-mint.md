# 矿工运行说明（不涉及发币）

本项目**不发币**，本文档也不对“代币奖励/质押收益/挖矿产出”做任何假设。

如果你在当前环境中要运行 Worker 节点，请重点关注以下内容：

### 1. 硬件与证明（Attestation）
- 使用受支持的 TEE 硬件（例如 Intel SGX），并在 BIOS 中启用。
- 按照环境要求配置 DCAP / PCCS 等证明服务，保证节点可以通过远程证明。

### 2. 稳定性与运维
- 保持节点长期在线，避免频繁重启。
- 通过 Kubernetes 观察 Pod 状态与日志，及时处理异常。

### 3. 费用/结算（以环境为准）
节点的费用、结算或补贴策略（如果存在）取决于你所使用的具体部署环境（内测/合作网络/私有部署）。请以官方公告或环境管理员说明为准。
