# 支持的设备

### CPU 支持

* Intel 第 8 代 (Cannon Lake) Core i3, i5, i7 和 i9 处理器
* Intel 第 9 代 (Cascade Lake) Core i3, i5, i7 和 i9 处理器
* Intel 第 10 代 (Comet Lake) Core i3, i5, i7 和 i9 处理器
* 第 2 代 Xeon Scalable 处理器 (Cascade Lake) 及以后的版本通常提供 SGX 功能。
